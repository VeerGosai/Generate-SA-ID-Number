How South African ID Numbers Work

The ID number format is: YYMMDDGSSSCAZ, where:

  • YYMMDD: Date of birth
  • G: Gender (0-4999 for females, 5000-9999 for males)
  • SSS: Daily sequence number
  • C: Citizenship (0 for SA citizens, 1 for permanent residents)
  • A: Race indicator (8 or 9)
  • Z: Checksum digit (calculated using the Luhn algorithm)
